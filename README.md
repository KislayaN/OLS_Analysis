### Title 
OLS Analysis Toolkit - A diagnostic module for regression behaviour, convergence dynamics, and model comparision across OLS variants.

## What Problem it solves 
Most regression implementations stop at coefficients and MSE, but this tool exposes convergence behaviour, scaling sensitivity, 
multicollinearity impact, and residual structure across multiple formulations.

## Features 
1. Model Coverage - OLS, Gradient Descent OLS, Linear Regression, Ridge, Lasso
2. Diagnostics - Residual plots, multicollinearity detection, correlation analysis
3. Performance Metrics - MSE, R2, convergence tracking, learning curves
4. Preprocessing Effects - Scaling impact
